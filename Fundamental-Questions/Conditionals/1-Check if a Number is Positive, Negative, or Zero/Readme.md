Problem Statement

Given an integer, determine whether it is positive, negative, or zero.

Input Format

The first line contains an integer T, the number of test cases.

Each of the next T lines contains an integer N.

Output Format

For each test case, print:

"Positive" if N > 0

"Negative" if N < 0

"Zero" if N == 0

Example

Input

3
5
-9
0


Output

Positive
Negative
Zero
